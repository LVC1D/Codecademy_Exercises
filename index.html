General Overview of HTML and CSS

What is HTML?

HTML is a declarative language; tells the computer what to do, without specifying the details of how exactly to do a certain action. HTML's code is based on elements, structured in a top-to-right, top-to-bottom manner, without anything fancy going on. The fancy stuff comes into play when we add CSS (more on that later).

Is HTML hard to write in?

HTML is one of those languages that some people consider a real coding language, and other - not so much. In my personal opinion, since elements are executable, and that semantics exist in writing out the code properly - that's a coding language in my books. However, keep in mind that no website or web app would exist without HTML. At all!

All about CSS


How do you CSS?

CSS is a natural "spouse" of HTML that is responsible for making sure the whole webpage doesn't look bland and boring. This normally involves a bunch of aspects, such as:

  • Changing typefaces and anything related to it
  • Adjusting colors of the element (and their background)
  • Positioning elements in a way that is outside of the typical HTML flow
  • and much more!...
